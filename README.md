# IoT Observability & Monitoring Stack
A comprehensive monitoring solution for distributed Edge Gateways to ensure high availability and proactive maintenance.

### Tech Stack:
- **Prometheus:** Time-series database for metric collection.
- **Grafana:** Advanced dashboarding for real-time telemetry.
- **Alertmanager:** Rule-based alerting for system downtime.

### Features:
- **Health Checks:** Shell scripts for automated container monitoring.
- **Metrics:** Tracking MQTT latency, CPU/RAM usage, and message throughput.
- **SLA Management:** Designed to maintain 99.5% system uptime for large-scale deployments.
